MDMTrackerManager

Introduction

MoodMe Tracker Object

Discussion

You can instantiate it as singletone or multiple instances to use separately. MDMTrackerManager is the main tracker object that takes in input an image and returns tracked data: landmarks, vertices and transformation matrix

Use CLAHE equalization 

@property (nonatomic) BOOL useCLAHE;
Discussion

Set this flag On to use CLAHE algorithm. CLAHE will equalize scene lights in order to improve tracking precision, but will reduce tracking performance. @remark Setting On can cause performance loss on some devices
